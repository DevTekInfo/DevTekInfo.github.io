RUST KeyBinds

Prevent Rust from defaulting your custom keybinds

  1. 1. Right click "key.cfg" and go to properties. Once the properties are opened select the "read only" check box.
  2. 2. Right click "key.cfg" select choose default program.
  3. 3. Deselct "always use this program to open this file type" and select word pad.
  4. 4. (Please Note: You MUST un-check this everytime you want to add or remove anything, the game will crash otherwise.)
  5. 5. Once "key.cfg" is open navigate to the bottom and place the commands you want to add.
  6. 6. When you have all the binds you want entered, click file at the top left and click save.
  7. 7. Close out of wordpad, Navigate back to C:\Program Files (x86)\Steam\steamapps\common\Rust\cfg
  8. 8. Right click "key.cfg" and go to properties. Once the properties are opened select the "read only" check box.
